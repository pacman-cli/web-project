## How to View

1. **HTML Code**: Open `index.html` within any screen folder directly in a web browser to view the interactive UI. The pages render utilizing the official Tailwind CSS CDN configuration, Inter typography, and Material Symbols.
2. **Screenshots**: Open `screenshot.png` in each folder to see the original reference mockup from the design project.
